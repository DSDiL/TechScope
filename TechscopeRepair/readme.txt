# TechScope
Repair part of a 8 part java web application for a technical equipment servicing company called Techscope.
In here employees add repair details to the system using their appointments.
They can search for both on going and completed repairs using the repair ID.
They can update and delete those accordingly and if the work is finished add it as completed.
They can get monthly reports on completed repairs.

how to run - 
      * Import sql file to database.
      * Add database password in DBconnect.class file
      * Start running from repairHome.jsp
